# VoiceVis 

VoiceVis is an application to visualize and highlight the gender aspects of voice. 

In short, the audio is analyzed using Opensmile [[2]](docs/70_references.md) and formant/pitch ratios are compared to target ratios. More details are available in [Methology](docs/20_methology.md).

How to use this application is described in [Usage](docs/10_usage.md).
